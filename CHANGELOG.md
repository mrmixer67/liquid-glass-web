# Changelog

All notable changes are documented here. Versions follow semantic versioning.

## 0.0.1 — 2026-07-13

- Initial public release of the standalone `liquid-glass.js` library.
- Live DOM backdrop and WebGL texture rendering modes.
- Configurable refraction, blur, tint, geometry, border and shadow.
- Pointer, touch and keyboard dragging without a pressed visual state.
- Selector- and element-based distortion exclusions.
- Automatic tests, GitHub Pages deployment and tagged GitHub Releases.
